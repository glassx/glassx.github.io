Ubuntu下搭建hexo

一、安装nodejs 与 npm

首先声明，博主目前主要使用Ubuntu进行Android开发，然后进行一些简单的日常使用，看到hexo官网上说要先安装nodejs和npm，首先想到的是去nodejs官网，但是下载下来解压之后，一脸懵逼，不知道怎么操作了，然后去网上搜搜怎么安装，各种五花八门的，不得要领，遂又去官网上找找解决方法，这里有个教训，这玩意儿一般还是在官网上找找，靠谱些，google上的不一定是最新的也不一定是最适合你的。我最后找到的官方指引在这里，有针对各种系统的安装说明，为了节约大家去看文档的时间，以下还是列出ubuntu环境下的安装过程：

在终端输入：

curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -

当然，如果提示你还未安装curl，则安装即可：

sudo apt install curl

完成之后，再在终端输入：

sudo apt-get install -y nodejs

最后，输入以下命令检测是否安装成功 ：

nodejs -v
v8.11.1
npm -v
5.6.0

如果显示这些版本号，说明安装成功。

二、安装hexo

有了前面的教训，这次还是首先找hexo的官方介绍，官方说明里面很详细地介绍了安装过程：

1、安装hexo

npm install -g hexo-cli

2、初始化hexo
在终端cd到你想要存放blog的目录，执行

hexo init BLOG (这里BLOG是你想要新建的存放blog的文件夹名称)
cd BLOG //进入目录

3、安装依赖

npm install

三、完成git配置

申请github账号，并且在本地电脑上安装git ，并且将本地仓库与github的远程仓库关联起来，详细过程此处略，另寻答案。
附：github提供的ssh设定：https://help.github.com/articles/connecting-to-github-with-ssh/
